# 🚀 JavaScript Mini Projects

A collection of beginner-to-intermediate JavaScript projects built while learning Web Development and strengthening DOM Manipulation, Event Handling, and JavaScript fundamentals.

## 📌 About

This repository contains hands-on projects developed using **HTML**, **CSS**, and **Vanilla JavaScript**. The goal of these projects is to practice core JavaScript concepts and build a strong foundation before moving to modern frameworks like React.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* DOM Manipulation
* Event Handling

## 📂 Projects Included

### 🎨 Background Color Changer

Randomly changes the webpage background color with a button click.

**Concepts Practiced:**

* Arrays
* Math.random()
* Event Listeners
* DOM Manipulation

---

### 🔢 Counter App

A simple counter application with increase, decrease, and reset functionality.

**Concepts Practiced:**

* Variables
* Event Listeners
* DOM Updates
* Functions

---

### 🎲 Dice Game

A fun dice rolling game that generates random dice values.

**Concepts Practiced:**

* Random Number Generation
* Conditional Logic
* DOM Manipulation

---

### ✅ Todo App

A task management application where users can add, complete, and remove tasks.

**Features:**

* Add Tasks
* Delete Tasks
* Mark Tasks as Complete

**Concepts Practiced:**

* Arrays
* Dynamic DOM Creation
* Event Handling
* Input Management
